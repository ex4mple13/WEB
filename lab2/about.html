About iLearn Project

This is a static web application called 'iLearn', developed by Oleksandr AkimovKV-23 for a laboratory assignment in the 'Web-design' course.

This project demonstrates the skills in creating responsive user interfaces and utilizes HTML5, CSS3, and the Bootstrap 5 framework.

Key Features and Pages:

  • Word Learning: The main page showcasing vocabulary learning and testing modes.
  • Registration: A form for new users to sign up for an account.
  • Login: A form for existing users to log into their account.
  • Profile: Displays the user's personal information.
  • About: Provides information about the project itself (this page).
